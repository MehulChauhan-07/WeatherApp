ByteForge Frontend - Detailed Overview & Setup Guide
The frontend of ByteForge is designed to provide a seamless, beginner-friendly experience for learning Java online. It is developed using React (Vite + TypeScript) with Tailwind CSS, DaisyUI, Herui, and ShadCN for a modern, aesthetic UI.

1️⃣ Features in Detail
🎨 User Interface & Design
ByteForge offers an intuitive and clean UI that is optimized for all devices.

    🔹 Tech Stack for UI
            React (Vite + TypeScript) → Ensures better performance and type safety.

            Tailwind CSS → Utility-first styling for rapid development.

            DaisyUI, Herui, ShadCN → UI component libraries for a polished, modern look.

            Framer Motion (Optional) → Smooth animations and transitions.

            Lucide-React Icons → High-quality icons for better visual representation.

    🔹 UI Components
            Navbar → Dynamic navigation with authentication-based links.

            Sidebar → Used for note-taking and editor settings.

            Button Components → Reusable buttons with different variants (primary, secondary, etc.).

            Modals & Dialogs → Used for login, settings, and error messages.

            Forms → Login, signup, and note creation.

      🖊️ Code Editor
      The platform includes an interactive Monaco Editor (VS Code's editor) to provide a real coding environment.

    🔹 Key Features
            Java Syntax Highlighting → Makes Java code easier to read.

            Auto-Completion → Suggests methods, variables, and Java keywords.

            Error Highlighting → Helps users identify mistakes in real-time.

            Theme Customization → Light/Dark mode for a comfortable coding experience.

            Run Button → Sends the Java code to the backend for execution.

    🔹 How Code Execution Works
            The user writes Java code inside Monaco Editor.

            When they press the "Run" button, the code is sent to the backend API.

            The backend runs the code inside a Docker container.

            The output is sent back and displayed in the frontend.

📝 Notes Keeper
    ByteForge allows users to take notes while coding.

    🔹 Key Features
            Create, Edit, Delete Notes → Helps users save learning material.

            Rich Text Formatting → Allows users to add headings, bold text, and lists.

            Auto-Save Feature → Saves notes automatically in the database.

            Search Notes → Quickly find previous notes.

🤖 AI Chatbot
    ByteForge includes an AI chatbot to assist learners with Java concepts.

    🔹 Key Features
          Text-Based Chatbot → Users can type Java-related queries.

          Instant Answers → Provides explanations, example code, and debugging help.

          Future Enhancements

          Voice-based interaction

          AI-powered code suggestions

🔑 Authentication System
    User authentication is implemented for secure access.

    🔹 Key Features
          JWT/OAuth Authentication → Secure login & session management.

          Google & GitHub Login (Planned Feature) → Social authentication for faster access.

          Session Persistence → Users stay logged in even after refreshing the page.

📱 Responsive Design
    ByteForge is fully responsive across desktops, tablets, and mobile devices.

    🔹 Mobile-Friendly Features
          Collapsible Navigation Bar

          Drawer Menu for Mobile

          Adaptive Code Editor for Small Screens
